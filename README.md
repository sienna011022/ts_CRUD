# BIRD_VIEW 김성윤 과제

## ✨기술 스택

> 해당 프로젝트를 수행하며 사용할 기술 스택입니다.

- NodeJs
- Express JS
- TypeScript
- TypeORM

## ✨기능

> 각 모듈 별 기능을 나열하였습니다

- [X] 회원을 등록한다
- [ ] 전체 회원을 조회한다

- [ ] 전체 게시물을 조회한다
- [ ] 전체 사진을 조회한다

- [ ] 회원이 게시물을 등록한다
- [ ] 회원의 게시물을 조회한다
- [ ] 회원이 게시물을 삭제한다

- [ ] 게시물을 수정한다
- [ ] 게시물을 삭제한다
- [ ] 게시물에 사진을 등록한다
- [ ] 게시물에 사진을 전체 삭제한다
- [ ] 특정 사진을 조회한다
- [ ] 특정 사진을 수정한다
- [ ] 특정 사진을 삭제한다
